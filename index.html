SubnetForge

Computes FLSM and VLSM subnet tables and auto-generates Cisco IOS CLI commands ready to paste into Packet Tracer.

Problem

Computing VLSM and FLSM tables by hand during networking labs was slow and error-prone. One wrong value broke the entire Packet Tracer configuration. Typing CLI commands manually after made it worse.

What I Built

An offline web app that takes an IP address and subnet requirements, computes the full routing table, and generates the exact Cisco IOS commands for copy-paste use. No backend, no install, runs in any browser.

Result

Eliminated manual computation errors in my own lab work. Supports Packet Tracer 5.3.3 and below for legacy labs, plus Fast Ethernet on newer versions.

Flashcard Reviewer

Offline flashcard app using the SM-2 spaced repetition algorithm with streaks, XP, and level-ups to build a daily study habit.

Problem

Physical flashcards made of bond paper were time-consuming to make and easy to lose. Moving to Anki or Gizmo meant hitting paywalls for basic features like creating and customizing cards.

What I Built

A fully offline single-file web app implementing the SM-2 spaced repetition algorithm. Cards are scheduled based on recall quality. Streaks, XP, and level-ups reward consistent daily use. Supports folder organization, import, export, and shuffle.

Result

Free to use, no account, no subscription. Runs in any browser from a single downloaded file. The motivation loop keeps me returning daily in a way Anki alone did not.

Arduino Uno C++ Motion Sensors LED Circuit

Drag Race Timing System

Arduino-based drag race system with LED countdown lights and motion sensors that detect vehicle start and stop events on a physical track.

Problem

Applying microprocessor subject learnings in a tangible, real-world context. The goal was to move from theory to a working physical system controlled entirely by code.

What I Built

A physical drag race track with an Arduino Uno controlling an LED light tree (red, yellow, green sequence) and motion sensors at start and finish lines. Sensors detect when a vehicle crosses the line and the system records the event. Programmed in C++ using Arduino IDE.

Result

A fully functional physical build that demonstrates microcontroller logic, sensor interfacing, PWM LED control, and real-time event detection working together.
